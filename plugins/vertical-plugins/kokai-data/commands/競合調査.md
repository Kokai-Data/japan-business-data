# /競合調査 (alias of /competitor-brief)

日本企業の 3-tier 競合 brief を生成 (Tier 1: 概観 / Tier 2: 規模 / Tier 3: 認定)。法人番号から gBizINFO 企業 profile + 国税庁 法人番号 異動履歴 + 関連法人 signal を citation 付きで構成。4 階層 authority strip + 士業 boundary disclaimer 付き。

使い方: `/競合調査 <13-digit 法人番号>` (例: `/競合調査 1234567890123`)

英語 alias: `/competitor-brief` (同一機能、後方互換維持)

詳細 workflow は [competitor-brief-jp agent](../agents/competitor-brief-jp-agent.md) 参照。

## Boundary

Output は signal / 確認材料 / context — 競合判断 / 戦略判断 は別途専門家 (中小企業診断士 / 経営コンサルタント) 確認前提。Kokai は判定者ではなく signal 提示者。
