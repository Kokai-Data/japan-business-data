---
description: 通関分類の確認材料を出典つきで整理（Kokai Data 貿易）。Kokai は確認材料・signal の提示者です。最終確認は通関士・税関へ。
---

製品説明・材質・用途・HS コード（任意）から、通関分類に関する公的データの **確認材料 / 分類候補 / 出典つき signal** を整理してください。Kokai Data の `customs_classification_advisory` tool、または必要に応じて `customs_*` tools を呼び、出典とともに結果を提示します。

表現規則:
- 断定的な結論表現や推奨表現を避ける。
- 「分類候補」「確認材料」「signal」「出典」に寄せる。
- 最終的な HS 分類・関税率適用・通関手続は、通関士・税関で確認が必要であることを明記する。
