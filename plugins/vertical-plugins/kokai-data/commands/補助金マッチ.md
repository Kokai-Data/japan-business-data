# /補助金マッチ (alias of /subsidy-fit)

日本企業と特定の補助金の組み合わせに対して、3 軸 (地域 / 業種 / 規模) で fit signal を生成。J-Grants subsidy 詳細 + gBizINFO 企業 profile を citation 付きで照合、likely_relevant / unclear / likely_not_relevant の signal 形式で提示。4 階層 authority strip + 士業 boundary disclaimer 付き。

使い方: `/補助金マッチ <13-digit 法人番号> <J-Grants subsidy_id>` (例: `/補助金マッチ 1234567890123 a0WJ200000XXXXXMA1`)

英語 alias: `/subsidy-fit` (同一機能、後方互換維持)

詳細 workflow は [subsidy-fit-jp agent](../../../agent-plugins/subsidy-fit-jp/agents/subsidy-fit-jp-agent.md) 参照。

## Boundary

Output は **fit signal / 確認材料 only** — 申請可否判断 / 適格性判断 / 申請戦略 は **行政書士 / 中小企業診断士** 確認前提。Kokai は判定者ではなく signal 提示者。
