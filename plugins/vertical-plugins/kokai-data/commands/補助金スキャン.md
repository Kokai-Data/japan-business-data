# /補助金スキャン (alias of /subsidy-landscape)

業種または用途テーマで日本補助金 (J-Grants) の Top-N landscape を scan。J-Grants 全件から該当 keyword 補助金を citation 付きで抽出、signal overlay 形式で提示 (filter ではなく signal classification、47 都道府県並列で返す)。4 階層 authority strip + 士業 boundary disclaimer 付き。

使い方: `/補助金スキャン <keyword>` (例: `/補助金スキャン DX`, `/補助金スキャン 事業承継`, `/補助金スキャン 製造業`)

英語 alias: `/subsidy-landscape` (同一機能、後方互換維持)

詳細 workflow は [subsidy-landscape-jp agent](../agents/subsidy-landscape-jp-agent.md) 参照。

## Boundary

Output は landscape signal / 確認材料 only — 申請可否判断 / 採択可能性 / 戦略判断 は **行政書士 / 中小企業診断士** 確認前提。Kokai は判定者ではなく signal 提示者。
