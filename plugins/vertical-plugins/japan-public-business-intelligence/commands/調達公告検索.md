# /調達公告検索 (alias of /procurement-discovery)

日本の政府調達公告 (官公需情報ポータル、中小企業庁) を検索。Phase 1 scope = CFT (Call for Tender / 公告) discovery only — project name / 発注機関 / 業種分類 / 入札方式 / CFT issue date を取得。Phase 2 (落札結果 / 受注実績 / 価格 signal) は official API 非開示のため deferred。4 階層 authority strip + 行政書士 / 中小企業診断士 / 弁護士 boundary disclaimer 付き。

使い方:

- `/調達公告検索 <keyword>` (例: `/調達公告検索 システム開発`)
- `/調達公告検索 <発注機関>` (例: `/調達公告検索 中小企業庁`)
- `/調達公告検索 <自治体コード>` (例: `/調達公告検索 131016`)

英語 alias: `/procurement-discovery` (同一機能、後方互換維持)

詳細 workflow は [procurement-discovery-jp agent](../../../agent-plugins/procurement-discovery-jp/agents/procurement-discovery-jp-agent.md) 参照。

## Boundary

Output は CFT discovery signal / 確認材料 only — 入札判断 / 適格性判断 / 法的判断 は **行政書士 / 中小企業診断士 / 弁護士** 確認前提。Kokai は判定者ではなく signal 提示者。
