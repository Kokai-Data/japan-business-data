# /法令調査 (alias of /legal-research)

日本法令 (法律 / 政令 / 省令) を e-Gov 法令検索 (デジタル庁) で検索 or 特定 law_id で条文 verbatim + 章・条構造を取得。4 階層 authority strip + 弁護士 / 行政書士 / 司法書士 boundary disclaimer 付き。

使い方:

- `/法令調査 <keyword>` — keyword 検索 (例: `/法令調査 中小企業`, `/法令調査 個人情報保護`)
- `/法令調査 <law_id>` — 条文 verbatim 取得 (例: `/法令調査 338AC0000000154`)

英語 alias: `/legal-research` (同一機能、後方互換維持)

詳細 workflow は [legal-research-jp agent](../../../agent-plugins/legal-research-jp/agents/legal-research-jp-agent.md) 参照。

## Boundary

Output は **公式 statute text verbatim** — 法令解釈 / 法的助言 / 適用判定 / 訴訟戦略 は **弁護士 / 行政書士 / 司法書士** 確認前提。Kokai は判定者ではなく signal 提示者。
