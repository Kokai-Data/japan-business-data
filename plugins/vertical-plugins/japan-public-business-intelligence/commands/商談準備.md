# /商談準備 (alias of /meeting-prep)

日本企業の商談前 brief を 1 ページで生成。法人番号 (13 桁) または会社名で gBizINFO + 国税庁 法人番号 + J-Grants 補助金 landscape signal を citation 付きで引用。4 階層 authority strip + 士業 boundary disclaimer 付き。

使い方:

- `/商談準備 <13-digit 法人番号>` — direct (最速)
- `/商談準備 <会社名>` — 候補 search → user confirm

英語 alias: `/meeting-prep` (同一機能、後方互換維持)

詳細 workflow は [meeting-prep-jp agent](../../../agent-plugins/meeting-prep-jp/agents/meeting-prep-jp-agent.md) 参照。

## Boundary

Output は signal / 確認材料 / context — 商談判断 / 投資判断 / 適格性判断 は別途専門家 (中小企業診断士 / 税理士 / 弁護士 等) 確認前提。Kokai は判定者ではなく signal 提示者。
