# /補助金マッチ (alias of /subsidy-fit)

`subsidy-fit-jp-agent` を invoke、日本企業 × 補助金の 3 軸 (地域 / 業種 / 規模) fit signal を生成。

See [agent system prompt](../agents/subsidy-fit-jp-agent.md) for full workflow.

使い方: `/補助金マッチ <13-digit 法人番号> <J-Grants subsidy_id>`

英語 alias: `/subsidy-fit` (同一機能、後方互換維持)

Returns: fit signal (likely_relevant / unclear / likely_not_relevant) + 4 階層 authority strip + 士業 boundary disclaimer (申請可否は行政書士 / 中小企業診断士 確認前提)。
