# /調達公告検索 (alias of /procurement-discovery)

`procurement-discovery-jp-agent` を invoke、日本の政府調達公告 (官公需情報ポータル、中小企業庁) を検索 (Phase 1 = CFT discovery only)。

See [agent system prompt](../agents/procurement-discovery-jp-agent.md) for full workflow.

使い方:

- `/調達公告検索 <keyword>` (例: `/調達公告検索 システム開発`)
- `/調達公告検索 <発注機関>` (例: `/調達公告検索 中小企業庁`)
- `/調達公告検索 <自治体コード>` (例: `/調達公告検索 131016`)

英語 alias: `/procurement-discovery` (同一機能、後方互換維持)

Returns: cited CFT discovery list + Phase 1 scope reminder + 4 階層 authority strip + 行政書士 / 中小企業診断士 / 弁護士 boundary disclaimer。
