# /法令調査 (alias of /legal-research)

`legal-research-jp-agent` を invoke、日本法令 (e-Gov 法令検索、デジタル庁) を keyword で検索 or 法令 ID で条文 verbatim 取得。

See [agent system prompt](../agents/legal-research-jp-agent.md) for full workflow.

使い方:

- `/法令調査 <keyword>` — keyword 検索
- `/法令調査 <law_id>` — 条文 verbatim 取得

英語 alias: `/legal-research` (同一機能、後方互換維持)

Returns: cited 法令 list OR verbatim 条文 + 4 階層 authority strip + 弁護士 / 行政書士 / 司法書士 boundary disclaimer。
