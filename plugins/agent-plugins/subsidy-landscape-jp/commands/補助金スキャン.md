# /補助金スキャン (alias of /subsidy-landscape)

`subsidy-landscape-jp-agent` を invoke、業種・用途テーマで日本補助金 (J-Grants) の Top-N landscape を scan。

See [agent system prompt](../agents/subsidy-landscape-jp-agent.md) for full workflow.

使い方: `/補助金スキャン <keyword>` (例: `/補助金スキャン DX`, `/補助金スキャン 事業承継`)

英語 alias: `/subsidy-landscape` (同一機能、後方互換維持)

Returns: cited Top-N subsidy landscape (signal overlay 形式) + 4 階層 authority strip + 士業 boundary disclaimer (申請可否は行政書士 / 中小企業診断士 確認前提)。
